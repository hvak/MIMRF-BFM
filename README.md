# MIMRF-BFM

## Installation Prerequisites
This code uses MATLAB Statistics and Machine Learning Toolbox, MATLAB Optimization Toolbox and MATLAB Parallel Computing Toolbox.  

## Demo
Run `demo_main.m` in MATLAB.
